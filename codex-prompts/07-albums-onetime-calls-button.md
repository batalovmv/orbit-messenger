# Задача: Альбомы (grouped media), One-time медиа, скрыть кнопку звонка

## Роль и поведение
Ты — senior frontend-разработчик проекта Orbit Messenger (форк Telegram Web A). Работай автономно, принимай решения сам. После каждого блока изменений перечитай код и убедись в корректности.

## Часть 1: Альбомы (grouped media)

### Контекст
В Telegram фото/видео отправленные вместе группируются в альбом — красивая сетка из 2-10 медиа в одном блоке. В Orbit бэкенд уже поддерживает `grouped_id` — сообщения с одинаковым `grouped_id` принадлежат одному альбому. Но фронтенд отображает их как отдельные сообщения.

### Что нужно сделать
1. **Найти как TG Web A группирует сообщения** — в оригинальном коде уже есть компонент `Album`. Найди его:
   - `web/src/components/middle/message/Album.tsx` или аналог
   - Поищи `grouped`, `album`, `groupedId` в `web/src/components/middle/`

2. **Проверить маппинг `grouped_id` из Saturn** — в `web/src/api/saturn/` найди где Saturn response маппится в TG message объект. Поле `grouped_id` из Saturn должно маппиться в `groupedId` TG Web A message.

3. **Обеспечить группировку** — в message list рендеринге (`MessageList.tsx` или `Message.tsx`) должна быть логика которая группирует сообщения с одинаковым `groupedId` и рендерит их через Album компонент.

### Визуальный результат (как в Telegram)
- 2 фото: 2 колонки, одинаковая высота
- 3 фото: 2 сверху + 1 снизу (или 1 + 2)
- 4+ фото: grid layout с автоподбором
- Все фото в альбоме имеют общий скруглённый контейнер (`border-radius: 0.75rem`)
- Время показывается один раз на последнем фото альбома
- Gap между фото: `2px`

## Часть 2: One-time медиа

### Контекст
В Telegram при отправке фото/видео можно включить "одноразовый просмотр" — получатель может открыть медиа только один раз. Бэкенд Orbit уже поддерживает поле `is_one_time` на сообщениях. На фронтенде нужна кнопка toggle.

### Что нужно сделать
1. **Найти UI отправки медиа** — `web/src/components/middle/composer/AttachMenu.tsx` или `SendMediaModal` или аналог
2. **Добавить toggle-кнопку** — иконка одноразового просмотра (таймер/число 1) рядом с кнопкой отправки в модале превью медиа
3. **Передать флаг** — при отправке сообщения с медиа, если toggle включён, добавить `is_one_time: true` в body запроса
4. **Отображение** — сообщения с `is_one_time` должны показывать размытое превью с иконкой "1" поверх. После просмотра — "Opened" текст.

### Визуальный стиль кнопки
- Иконка: кружок с цифрой "1" внутри (или timer icon)
- Позиция: в toolbar модала превью, рядом с другими опциями
- Состояния: неактивный (серый), активный (синий/accent)
- CSS: `width: 2rem; height: 2rem; border-radius: 50%;`
